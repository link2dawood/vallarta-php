  FEATURE #1: WhatsApp Invoice 📱
════════════════════════════════════════════════════════════

WHAT IT DOES:
Creates a plain-text invoice that you can copy and paste
directly into WhatsApp messages to customers.

WHERE TO FIND IT:
Go to: finalize_order.php (when finalizing any order)
Scroll down to the green "WhatsApp Invoice" section

HOW TO USE:
1. Open the finalize order page for any order
2. See the formatted invoice in the text box
3. Click "Copy to Clipboard" button
4. Open WhatsApp
5. Paste and send to customer

WHAT'S INCLUDED:
✓ All order details (same as PDF receipt)
✓ Products with prices (MXN & USD)
✓ Customer information
✓ Delivery address and ETA
✓ Payment instructions
✓ Client ID and Receipt ID
✓ Complimentary items
✓ Professional formatting with emojis

WHY IT'S BETTER:
• Saves 2-3 minutes per order
• No PDF download/upload needed
• Customers get instant confirmation
• Professional WhatsApp formatting
• Works on all devices

  FEATURE #2: Brand Management System 🏷️
════════════════════════════════════════════════════════════

WHAT IT DOES:
Adds a complete "Brand" system (like Categories and Strains)
for organizing products by cannabis brand names.

WHERE TO FIND IT:

ADMIN SIDE:
• Manage Brands: admin/addbrand.php
• Add/Edit Products: Now has "Brand" dropdown

CUSTOMER SIDE:
• Navbar: New "Brand" menu between Strains and Category
• Filter: movie.php?brand=X

HOW TO USE (ADMIN):
1. Go to admin/addbrand.php
2. Add brand names (Cookies, Stiiizy, etc.)
3. When adding products, select brand from dropdown
4. Brands appear in website navigation automatically

HOW IT WORKS (CUSTOMERS):
1. Click "Brand" in navigation menu
2. See list of all brands
3. Click brand name to see all products from that brand
4. Works with pagination
